回程随想

发表于 2019-11-07   |   分类于 随笔

杭州三天的培训，见了同学、曾经以及现在的同事，见了行业内的从业人员，上次参加这样的考试还是四年前的西安，此时与彼时心境已大不同，现在的这次聚会，就像当年在保定结业的酒宴上，大家各自笑笑，笑着祝福前程似锦，只是现在的这次没有酒会，只有大家挥挥手再见，笑着说See you，一转身笑容留在身后，只剩下沉重的心情伴随着我。

进站，路口都有检票口、服务台、餐饮、厕所的清晰指示，可是我的路上没有指示，只有一盏油灯，仅照亮我的脚下。坐上列车我就知道四小时后我会到达曲阜，我们自己说，正因为未来是不确定的才精彩，可是火车早来或者晚点都不能给我带来快乐和惊喜。早了，我没赶上，晚了，我需要等。不要说，早来了，就能早点到家，晚点，可能有时间吃个晚饭，不至于饿着。我从不认为这种自我安慰的东西是真正的解决之道。

如果明天是确定的，如果我们身边的所有事情就像DCS内部的逻辑一样被编程好了，只等条件的具备触发合适的下一步，人生就是一个APS，我们所有的想法和思想都是所有条件决定的触发，就像我要写这篇文章，条件一：是因为我在火车上，我有4个小时的时间来思考这件事情，条件二：上车前看到了路牌，条件三：我听到赵传在唱，如果有一天我迷失风雨中；条件四：我因为背了两天的考试题，有一点的累；条件五，条件六~~~等等，所有的合适的、已知的、未知的条件促成了我现在的这种想法，那到底哪一个想法是我的真实想法，那到底哪一个意识是我的真实自我，都说我们与机器人的区别是我们有自我意识，那我真的有吗？

如果明天是不确定，如果明天的事情就像今晚的火车，可能会晚点，可能会早到，我问L：亲爱的，你说未来不定，是该开心还是该悲伤？L告诉我：未来不定才会有更多的变好的可能啊。可是亲爱的，如果想让明天变好该怎么办，需要我们努力吗，如果我们努力就一定会变好么，可是诸多的事情证明这不是必然的因果，因果只不过是我们祖先观察整个世界所总结出来的经验而已，如果深究，因果的说法靠不靠得住还不一定，但是我知道如果让我们扔掉因果的想法是会出乱子的，且不去想因果，努力不一定导致明天会变得更好，比如我努力呼吸，明天不会变得更好，缘木求鱼，我们知道是可笑的，可是你怎么知道，今天你努力学英语就不是在水中捞月，我们每一个人可能都希望明天会变得更好，并且这个不确定的明天一定会到来，那我们该做点什么？

耳机中赵传在唱，“试着找寻自我，别再为爱蹉跎，只是爱要怎么说出口。”我们在试着找寻自我，别再明天担忧，只是如何改变我们生活。活了三十三年，见到了大约一万两千个明天，这些构成了我，过往我就像在船坞里面修了又修的忒修斯，终于在三十三周岁之前的两天的今天晚上我觉得停住这一切，想一想我该换那一块甲板，还是该换螺旋桨。

就像上面说的我们不知道如何做才不是竹篮打水，生活不像我的工作，如果我检查了所有的可能性，做好了静态仿真实验，按照合理的流程我就能得到预期的结果，并且我手中的油灯无法让我检查所有的可能性，生活更不能做仿真实验，我想从哲学中找出问题的所在，直到现在我都没有找到，从笛卡尔的，我思故我在，到休谟的实用主义思想，休谟告诉我，所谓因果关系之间其实存在一个信心或盼望的跳跃，不是知识，有个简单的例子，每天火鸡都得到了美味的食物，火鸡得出结论，每天都能得到食物，明天也是，可这只火鸡不知道明天是感恩节。想到这些，我有些悲观，当然后来康德对休谟的因果论给予了反驳，但是仍不能缓解我现在的悲观。我告诉L现在的心情，需要些励志的鸡汤，也许火鸡的也行。终究我陷入了我上面所说的不真正的解决之道。

如果哲学中的问题过于复杂，我想到了过去的历史，万历元年张居正上疏实行‘考成法’的时候，他可曾想到他的改革带来万历中兴，又可曾想到十年后的病逝以及改革又都废除。又当大明王朝的最后一位天子在景山的时候他又可曾想到有这一天。所有的历史事件太多，就像《桃花扇》里说的，眼见他高楼起，眼见他眼宾客，眼见他楼塌了。我在想，群星璀璨的众多人物中，都无法避免或好、或坏的明天，无一例外。

如果没有以上如果，我就是我，该怎么办？

杭州-曲阜 | 2019/11

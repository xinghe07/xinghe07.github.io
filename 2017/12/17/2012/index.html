2012

发表于 2017-12-17   |   分类于 随笔


我总是讲不清一个故事，逻辑的混乱在我的生活里坚强的生活着，下决心站在2012年夏日的路口回忆曾经的过往，可片段的片段需要我努力的串起，在破碎的记忆底片中，在摇曳的镜头里。

在K52上挥手告别盛夏的济南，告别熟悉的街道和石兄踏上保定的长途车，在偌大的车站里没有送行的伤感，只有对未来那个陌生城市的向往，追逐自己遥不可及并且做了那么多年的梦，来不及回头看看，窗外已是一望无际的原野，到了只有在地理课上听说过的华北平原，看到了直隶总督府的威严。

一个月的培训，见到了未来的同事，怀揣各种想法的每个人，在自己的世界里活得悠然，然后在培训结业时拿到了自己这辈子赚到的第一个月的工资—800元，一个月我争取过争先的报告，我有过为别人做嫁衣的无奈，在结业的那天酒宴上我们各自笑笑，笑着祝福前程似锦，在心里默默祝愿自己的未来，不似锦，不落魄。

8月1日，下午，站在成安电厂的门前，夕阳的余辉映红了高高的烟囱， 湛蓝湛蓝的天空里蜷着如丝绒般的云，微风吹着玉米秆的味道，少了济南的炎热，少了哥们”走，吃饭”的呼喊，看着篮球架下一遍遍投球的少年，这个地方投进的是不是我的梦想。

热工专业是我选择到现在的七年里感觉还算为数不多的正确的一件事，当然在今年选择了正确的人来陪我后半辈子是我最正确的选择，这是后话。热控是一个需要思考的专业，是一个磨练你性格的专业，容不得你半点鲁莽，容不得你半点粗心，基础的基础是最重要的，0和1的区别是天堂和地域的差别，同样简单的一根连线会是整个事件的成功和失败的关键，感谢我的师父，给我太多小心，注意，切忌的教导，检修班一年半的汗水，是我难忘的一年，在锅炉房外看见一点点亮起来的黑夜，看见升起的朝阳，看见引风机的叶轮就位和兄弟们疲惫的脸庞。多年以后还是记得那么清晰，曾经怀疑这就是我想要的吗，总是在一遍遍的劝自己，天降将大任。。。

多年以来检修班的酒桌上总少不了一个人的话题，千言万语，轮回百遍，风雨不变，成安在改变，人也在改变，改变的展览草草收场，真真切切感受到可能是我吧，那么多的镜头在我的相机里一点点的变化，在我记忆的底片上，渐渐的变化，当时不曾念，回即一想，百念丛生，时光荏苒。

2010年，本命年，至今，是我经历过最坎坷的一年，诸多不顺，现在回想起来，五味陈杂，可这一年教会了我很多以及很多很多不能说的事宜。这一年一个人到海边，看日出，看大海，自己从市区走到灯塔，东方已亮，海边有架好的相机，牵手期盼的情侣，我带着落寞的身影看着太阳从海面跳出来，没来得及按快门我，听着他们开心的笑声，呆呆的想，我这是怎么了，海水冲着细沙，来来回回的画面，想起我说出去散散心的时候，父母无奈的叹息，收起简单的行囊，回到成安，回到这个开始的开始。

爆竹声中一岁除，在年初的日子里我找回了L，上面提到的最正确的选择，那时的见面我连呼吸都曾反复练习，梦一样的相见。在那以前，就像歌里说的，曾经以为人生就这样了。记得L和我说，我找到我幸福了，我在我的状态更新，蝴蝶飞走了，真的认为她就是我的蝴蝶，摇曳在我的世界里。现在还是没有人回复那条状态，她是我的蝴蝶，没人懂。

好的开始奠定了忙碌的一年，回生产部，转专工，设备的考验，初入行的忙乱，就连贴个单据都要别人帮忙改好几遍，报个计划都不知道要买什么，检 修计划都不知道哪是重点，在慌乱中适应着，当然还有那个一天1200出的L。后半年特别忙，忙着检修，一次次的检修，打拼着不确定的未来，和L说，有着太多不确定的未来，怎么办，L说，只要我们确定就好了，慢慢来，你还年轻。那么简单的一句话让我释怀到现在，我现在还在慢慢来不着急有所作为，不着急都懂，在慢慢学习，一点点，一滴滴。

总有那么一个人让你收了心，总有那么一个人收了你半世流离，敛了半世癫狂，安安心心的陪她一辈子，简简单单。

入职场四年，算不得坎坷，不似锦，不落魄，慢慢找回自己，找正自己，换得心安。

2012年8月4日 于成安
